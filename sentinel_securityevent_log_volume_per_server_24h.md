# SecurityEvent Log Volume per Server (Last 24 Hours)

## Description

This Kusto Query Language (KQL) script queries the `SecurityEvent` table in Microsoft Sentinel to identify **the volume of Security Event logs generated by each server during the last 24 hours**.
It filters for **events generated within the past 24 hours**, calculates the **total billed data size per server**, converts the result to megabytes, and sorts the output to show **the servers generating the largest amount of log data**.

## Purpose

* **Log volume analysis:** Identify which servers generate the most Windows security event logs.
* **Cost monitoring:** Understand ingestion costs since `_BilledSize` reflects billable log data volume.
* **Source prioritization:** Detect systems producing unusually high logging activity.
* **Capacity planning:** Assist in estimating storage and ingestion requirements.

## Data Source

* **Log Table:** `SecurityEvent`
* **Key Fields Used:** `TimeGenerated`, `Computer`, `_BilledSize`

## KQL Query

```kusto
SecurityEvent
| where TimeGenerated > ago(24h)                      // Only include events from the last 24 hours
| summarize TotalSizeBytes = sum(_BilledSize) by ServerName = tostring(Computer)   // Sum billed data size per server
| extend TotalSizeMB = round(TotalSizeBytes / 1024 / 1024, 2)  // Convert bytes to MB
| project ServerName, TotalSizeMB                      // Show server and log volume
| order by TotalSizeMB desc                            // Sort by largest log volume
```

### Query Explanation

| Step                                 | Explanation                                                |
| ------------------------------------ | ---------------------------------------------------------- |
| `where TimeGenerated > ago(24h)`     | Limits the analysis to logs generated in the last 24 hours |
| `sum(_BilledSize)`                   | Calculates the total billable ingestion size               |
| `by ServerName = tostring(Computer)` | Groups the data by server hostname                         |
| `extend TotalSizeMB`                 | Converts the total size from bytes to megabytes            |
| `project`                            | Selects the fields shown in the result                     |
| `order by`                           | Displays servers generating the most logs first            |
